Project: Aplus Academy Website
Role: Frontend Developer / Designer
Overview: This repository contains the fully responsive Aplus Academy website, an educational platform designed for Ethiopian freshmen. The project demonstrates modern web development practices with:

Modular, reusable components including header, hero section, course cards, resources, registration form, and footer

Interactive features such as theme toggle, animated counters, and video showcases implemented with JavaScript

Mobile-first, responsive design using CSS for a consistent experience across devices

Semantic and well-structured HTML for readability, maintainability, and scalability

This project serves as a reference for educational web development and showcases practical frontend implementation skills.
