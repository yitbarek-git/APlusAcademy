# Aplus Academy Website

## 📌 Project Overview

The **Aplus Academy Website** is a fully responsive educational platform designed to support Ethiopian freshman university students. It provides structured learning resources, video tutorials, and academic support materials in a clear and accessible interface.

---

## 👨‍💻 Role

**Frontend Developer / UI Designer**

Responsible for:

* UI/UX design implementation
* Responsive layout development
* Interactive feature integration
* Performance and accessibility optimization

---

## 🧩 Key Features

### 1. Modular Architecture

* Reusable components:

  * Header & Navigation
  * Hero Section
  * Course Cards
  * Resource Section
  * Registration Form
  * Footer

### 2. Interactive Functionality

* Theme toggle (light/dark mode)
* Animated counters for statistics
* Embedded video showcase
* Dynamic UI behavior using JavaScript

### 3. Responsive Design

* Mobile-first approach
* Optimized for desktop, tablet, and mobile devices
* Consistent layout and spacing across screen sizes

### 4. Semantic HTML Structure

* Clean, readable markup
* SEO-friendly structure
* Improved accessibility and maintainability

---

## 🛠️ Technologies Used

* **HTML5** — semantic structure
* **CSS3** — layout, responsiveness, animations
* **JavaScript (Vanilla)** — interactivity and dynamic behavior
* **Font Awesome** — icons

---

This project demonstrates practical frontend development skills, including:
